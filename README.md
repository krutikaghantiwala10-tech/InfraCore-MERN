# InfraCore – Construction Contractor Management System

## 🚀 Features
- Admin, Contractor, User Panels
- Secure Authentication (Password Hashing)
- Email Response System
- Payment Integration
- CRUD Operations

## 🛠 Tech Stack
- MongoDB
- Express.js
- React
- Node.js

## 📌 Description
A full-stack MERN application developed to manage construction workflows and enable seamless communication between contractors and clients.

## ⚙️ Installation
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
